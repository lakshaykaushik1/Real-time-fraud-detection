## 📌 Overview
The Real-Time Fraud Detection System is a Machine Learning-based web application that detects fraudulent financial transactions instantly.

With the rapid growth of online payments and digital banking, fraud detection has become a critical challenge. This project provides a scalable and intelligent solution to detect suspicious transactions in real-time.

---

## 🎯 Problem Statement
Online transaction fraud is increasing rapidly.

Traditional fraud detection systems:
- Are slow and reactive
- Require manual review
- Fail to scale with high transaction volumes

There is a need for an automated, real-time, and accurate fraud detection system.

---

## 💡 Proposed Solution
This project uses Machine Learning models to:
- Analyze transaction patterns
- Detect anomalies
- Predict fraud in real-time

The system is deployed using **Streamlit** for an interactive user interface.

## ▶️ How to Run the Project
```bash
1. Clone the repository: git clone https://github.com/lakshaykaushik1/Real-time-fraud-detection.git
2. Navigate to the project folder: cd Real-time-fraud-detection
3. Install dependencies: pip install -r requirements.txt
4. Run the application: streamlit run app.py
```
